# BumperBot

A differential-drive mobile robot built with ROS 2 Jazzy, Gazebo Harmonic, and ros2_control.

## Features

- Differential drive robot
- Gazebo simulation
- RViz visualization
- ros2_control integration
- DiffDriveController
- JointGroupVelocityController
- Custom ROS 2 Python nodes
- Kinematics examples

## Requirements

- Ubuntu 24.04
- ROS 2 Jazzy
- Gazebo Harmonic
